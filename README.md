# nunumbers

A fast-paced arcade counting game built with Phaser 3. Test your visual counting skills through 30+ increasingly challenging levels!

## About

**nunumbers** is an arcade-style counting game where players must quickly identify and count animated sprites on screen before time runs out. With each level, the difficulty increases with more items, more variety, and less time to think!

## Features

- **30 Progressive Levels** - Start easy with 8 items, work your way up through increasingly challenging levels
- **Unique Animated Characters** - Discover and unlock new sprites as you progress
- **Discovery Gallery** - Track your progress and explore what you've unlocked
- **Ranking System** - Earn ranks from "space rookie" to "nunum god" based on your high score
- **Hidden Secrets** - Master the game to discover special surprises
- **Custom Pixel Art** - Retro-styled animated sprites and UI
- **Sound & Music** - Immersive audio with toggleable sound controls
- **Local High Score** - Track your best performance

## How to Play

1. **Open the game** - Simply open `index.html` in a web browser
2. **Navigate the menu** - Choose between Arcade mode and Discovery mode
3. **Arcade Mode**:
   - Count how many of a specific sprite appear on screen
   - Select the correct answer before time runs out
   - Progress through 30 levels (and beyond!)
   - Wrong answer or time out = game over
4. **Discovery Mode**:
   - View your collection
   - Check your high score and rank
   - Track your achievements

## Progression System

The game features 30 levels with progressively increasing difficulty. Every 5 levels, the challenge intensifies with more items on screen and greater variety. Can you master them all?

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No server setup required - runs entirely in the browser!

### Installation

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Click to start and enjoy!

### Running Locally

Simply double-click `index.html` or use a local server:

```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js (http-server)
npx http-server
```

Then navigate to `http://localhost:8000`
